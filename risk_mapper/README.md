# risk-mapper (v2)

Maps CTI outputs (TTPs, actor profiles) to risk frameworks via MITRE's published ATT&CK-to-controls mappings.

**Planned capabilities:**
- ATT&CK technique → NIST 800-53 / CIS Controls bridge
- Coverage gap scoring against an org's control baseline
- CTI-CMM maturity integration
- "Given these actors target your industry, here is your residual risk exposure"

Status: **backlog** — not yet implemented.
